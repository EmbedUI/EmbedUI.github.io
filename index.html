EmbedUI is an open source Project for enthusiasts, to speed-up & visualize UI development using Embedded Hardware on their PC/Laptop screens.

View Documentation

How it works

EmbedUI uses Arduino IDE (Download Arduino IDE). With the help of library API, you can display shapes, fonts and create actionable buttons to drive hardware actions through UI. View Instructions below to create EmbedUI

1. Download /Import library

Download and import EmbedUI ZIP library in the Arduino IDE by following this official documentaion

2. Code using Examples

View examples in the IDE to get started. Select an example of your choice, modify & upload it to you favourite board.

3. Download and Install Application

Download & Install the application. Allow the permissions to the application

4. Configure and connect

Set Baudrate and COM port to view the UI.
