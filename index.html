Fast - EmbedUI (beta)

Fast-EmbedUI is an open source Project, for enthusiasts to speed-up & visualize UI development using Embedded Hardware on their PC/Laptop screens.

View Documentation

How it works

Fast - EmbedUI uses Arduino IDE (Download Arduino IDE). with the help of library API, you can display shapes, fonts and create actionable buttons to drive hardware actions through UI. View Instructions to create embed UI below

1. Download /Import library

Import Fast-EmbedUI ZIP library in the arduino IDE by following steps given here.

2. Code using Examples

View examples in the IDE to get started after development burn code in the board.

3. Download and install Application

Download application allow the permissions to communicate with the board

4. Configure and connect

Set baud rate and comport to view the data on UI. Reset the board once connected to the application
